# spacha-cli
The YouTube superchat like image generator

## Installision
```
npm i -g spacha-cli
```

## Options
```
spacha-cli

  YouTube superchat like image generator. 

Synopsis

  $ spacha [options] <targetpath> 

Options

  -w, --width width                       Set a width of image. Default is 600      
  -h, --height height                     Set a height of image. Default is 300     
  --heightAutoFix [true|false]            Make tool decides a height of image.      
                                          Default is true                           
  -p, --price price                       Set a price of spacha.                    
  --customPriceString customPriceString   Set a custom string for price section.    
  -m, --message message                   Set a message of spacha.                  
  -t, --theme theme                       Set a theme of spacha. 'blue' |           
                                          'lightblue' | 'green' | 'yellow' |        
                                          'orange' | 'pink' | 'red'                 
  --user.name username                    Set a username of spacha.                 
  --user.image [filepath|URL]             Set a icon of spacha.                     
  --imgOption.square [true|false]         Make icon square.         
```
